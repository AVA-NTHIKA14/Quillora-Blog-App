## Overview  

Quillora is a modern content platform designed for creators to write, manage, and explore blogs with a clean and scalable architecture.  

It combines minimal UI design with real-world features such as authentication, dashboards, and dynamic content rendering.

---


## Features  

### Authentication  
- Secure login and signup  
- Session-based authentication  
- User-specific content access  

### Blog Management  
- Create and publish blogs  
- Structured writing interface  
- Dedicated writing route  

### Dashboard  
- Personalized user dashboard  
- Manage posts and activity  
- Clean UI for control  

### Feed System  
- Explore blogs from users  
- Dynamic content rendering  
- Smooth reading experience  

### Discussions  
- Community interaction support  
- Discussion-based engagement  

### API Routes  
- Backend powered by Next.js API routes  
- Scalable architecture  
- Clean separation of concerns  

---

## Tech Stack  

- Framework: Next.js (App Router)  
- Frontend: React  
- Authentication: Clerk  
- Backend: Next.js API Routes  
- Styling: CSS / Tailwind CSS  
- Deployment: Vercel
